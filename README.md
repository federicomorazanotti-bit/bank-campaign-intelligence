# Bank Term Deposit — Campaign Intelligence

Interactive dashboard analysing 45,211 outbound bank sales calls (2008–2010).
Built on the UCI Bank Marketing Dataset by Moro et al. (2014).

## Live dashboard
[View here](https://federicomorazanotti-bit.github.io/bank-campaign-intelligence)

## What's in this repo
- `index.html` — full interactive dashboard (7 tabs, Chart.js)
- `dashboard_walkthrough_v2.docx` — written walkthrough + Q&A

## Key findings
- Campaign conversion rate improved 15x over its lifetime (3.7% → 55.1%)
- Three pre-call CRM filters doubled the baseline conversion rate (11.7% → 21.9%)
- Logistic regression across 16 variables — revenue per call +87% on targeted pool
- Every figure verified from raw data. Limitations stated upfront.

## Data source
Moro, S., Cortez, P., & Rita, P. (2014). A data-driven approach to predict the success
of bank telemarketing. Decision Support Systems, 62, 22–31.
UCI ML Repository: https://archive.ics.uci.edu/dataset/222/bank+marketing
